\subsection{Detailed Description}
Adding fucurve editabillity functionality using the mouse and keyboard. 

Allows the user to interact with the bezier curve being drawn in one of three ways. ~\newline
Add a control point by using left key Delete a contorl point using a right click in a 5 pixel vicinity of the point Enable dragging mode by clicking the d key and then dragging using a held down left click.

Registers Clicks and responds to them by adding/deleting/dragging control points. 

Any operation is valid only is event is caused by a mouse press and not a mouse release

A mouse release signifies that dragging operation has ended if in dragging mode

Adds a new control point at the end of point list if the dragging mode is not enabled using the d key on a L\+E\+FT click.

It looks for and removes a point in a 5 pixel vicinity on a right click on the canvas.
