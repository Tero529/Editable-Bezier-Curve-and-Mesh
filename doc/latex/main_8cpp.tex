\subsection{Detailed Description}
Main Function for Curve and Surface Rendering. 

Open a bank blakc screen which provides space for an editable bezier curve. Left click to add a point, press d to enter into dragging mode, right click to remove a control point. upon clicking escape, drawing window is closed, surface of revolution of curve is obtained and saved as an O\+FF file.
