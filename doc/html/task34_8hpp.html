Detailed Description

Produces surface of revolution and OFF file of final curve.

Takes as input the control points of the final curve, as the user quits the editable application, uses them to create a surface of revolution for a curve sampled at t=0.1 and a 10 degree rotation sample This surface is stored in a vertex-face mesh data structure and ouputted in an OFF format The ouput should be redirected to desired file in the command prompt
